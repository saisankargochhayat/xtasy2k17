login form
  method : post
  action: "/users/login"
  parameters: 'login_email', 'login_password'

Sign Up form
  method : post
  action: "/users/register"
  parameters: 'name','password','email','college','year','phone','gender'

logout
  method : get
  uri : "/users/logout"

User account verification
  method: get
  uri : "/users/verify/:email/:hash"
  where ':id' is the value of _id field of the user's document in database

event creation in db(for admin page)
  method: post
  uri: "/events/create"
  parameters: 'event_id','name'
  where 'event_id' will be the unique id of the event
  set by the admin.

event registration
  method: put
  uri : "/events/register/:event_id"

event unregistration
  method: put
  uri : "/events/unregister/:event_id"

get an event document
  method: get
  uri: "events/:event_id"

NOTE:- use the awesome postman app to verify api routes
